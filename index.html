Политика конфиденциальности

Дата вступления в силу: 13 Марта 2026

1. Введение

Студия JoyfulSpark Studio ("мы") серьезно относится к защите данных пользователей. Данная Политика описывает работу нашего мобильного приложения при загрузке из RuStore, Huawei AppGallery, Google Play и других платформ.

2. Сбор технических данных и Идентификаторы

Мы не собираем персональные данные (ФИО, адрес). Однако для корректной работы приложения, аналитики и монетизации используются следующие рекламные и технические идентификаторы:

  • Google Advertising ID (GAID): Используется на устройствах с сервисами Google для персонализации рекламы.
  • Open Advertising ID (OAID): Используется на устройствах Huawei/Honor (HMS) и некоторых устройствах в РФ для работы рекламы без сервисов Google.
  • Android ID / UUID: Уникальный идентификатор устройства, используемый для сохранения прогресса и аналитики сбоев.
  • Данные об устройстве: Модель, версия ОС, язык системы, часовой пояс, IP-адрес (в обезличенном виде).

3. Сторонние сервисы

Приложение интегрирует SDK следующих партнеров, которые могут обрабатывать указанные выше идентификаторы:

  • Яндекс Реклама (Yandex Mobile Ads SDK) — основной партнер для монетизации в РФ/СНГ.
  • Huawei Mobile Services (HMS Core) — для работы на устройствах Huawei.
  • Unity Ads & Analytics — для игровой аналитики.
  • MyTracker / AppMetrica — для анализа стабильности работы приложения.

4. Безопасность детей

Наше приложение не предназначено для намеренного сбора данных о детях младше 13 лет.
